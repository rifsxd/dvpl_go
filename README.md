# dvpl-go
 A CLI Tool Coded In Golang To Convert WoTB ( Dava ) SmartDLC DVPL File Based On LZ4 Comrpession.

```

Usage :

 - Compression : "compress", "comp", "cp", "c"

 - Decompression : "decompress", "decomp", "dcp", "d"

 - Arguments : "--keep-originals", "-ko", "--keep-original"

 - Help : "--help", "-h"

```
